       11. CONTROLLERS OVERVIEW

       Container Controller (/app/scripts/controllers/container.js):
       - Manages tab navigation and language switching
       - Initializes with 6 tabs: NEW, INTRO, CV, WORKS, ARCHIVE, OTHER
       - Language toggle: switches between English/Finnish via changeLanguage()
       - Stores preference in $cookieStore under key com.laurakarki.preferredLanguage
       - Locale-aware routing (reconstructs paths like /intro/en → /intro/fi)
       - Listens to location changes to update selected tab

       News Controller (/app/scripts/controllers/news.js):
       - Simple controller with placeholder awesomeThings array
       - Injects: $scope, $translate, $cookieStore

       CV Controller (/app/scripts/controllers/cv.js):
       - Manages CV sections: EDUCATION, ARTISTIC_ACTIVITY, JOB_EXPERIENCE
       - Defines subsections for ARTISTIC_ACTIVITY
       - Key Feature: timeFor() function formats dates by granularity:
         - 'year': Year ranges (e.g., "2020-2022")
         - 'month': Month/year with ranges (e.g., "3/2020-6/2022")
         - 'day': Full dates (e.g., "15.3.2020")
         - Handles "continuing" flag for ongoing activities
       - Initializes with Finnish translations

       Works Controller (/app/scripts/controllers/works.js):
       - Initializes empty works array, fetches data
       - Fetches work names and descriptions from translations
       - Listens for $translateChangeEnd to refresh on language switch
       - Dynamic translation resolution via $translate service
       - Distinguishes between initial population (push) and updates (modify)

       Archive Controller (/app/scripts/controllers/archive.js):
       - Mirrors Works controller functionality
       - Fetches archive works data from translations
       - Handles language change events
       - Asynchronous promise resolution for translations
