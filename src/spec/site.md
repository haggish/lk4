
The site is located at http://laurakarki.com

## Overall site structure
The site's visuals and page structure equals that of the site located at http://laurakarki.com

The site has the following pages:
- Landing page
- Intro page
- CV page
- Works page
- Archive page
- Other page 

All pages have a header and footer.
The header has a navigation bar to all pages.
The footer has a toggle to switch languages.
The footer has links to social media.
The footer has a contact e-mail address.

## Landing page
- Landing page has a main photo
- Landing page has a list of news items
- One news item has a title, which is also a link, and news content, which can have links.

## Intro page
- Intro page has a biography

## CV page
- CV page has a list of education
- CV page has a list of artistic activity
  - artistic activity is divided to 
    - select private exhibitions
    - group exhibitions
    - poem performances
    - works in collections
    - commissioned works, public art
    - grants
    - prizes
    - memberships
    - residences
    - other
- CV page has a list of job experience
- all list items have two columns
  - time
    - can be range
    - can be date, then it is open range starting from the date
    - date can be year (example: 2022)
    - date can be month of year (example: 6/2022)
    - if date range is between months of same year, range is shown between months (example: 1-2/2024)
    - examples: 2020, 2020-2022, 1/2023-6/2026, 4/2022
  - content

## Works page
- Works page has a picture carousel of works, where each work has a title and information about the work

## Archive page
- Archive page is structurally similar to the works page.

## Other page
- Other page has miscellaneous information
