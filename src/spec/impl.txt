  Implementation Plan

  Visual Design (from lk2 source)

  - Background: white, max-width 730px centered
  - Header: fixed, 70px, border-bottom: 1px solid #e5e5e5, h1 "laura kärki" + subtitle
  - Footer: fixed, 80px, border-top: 1px solid #e5e5e5, language toggle left / social links + email right
  - Accent color: pink — links, active nav pill background
  - Nav: Bootstrap nav-pills, active tab = pink, centered on mobile
  - Fade gradients: 20px fade-in below header, fade-out above footer
  - Carousel: 70vh height, object-fit: contain

  Build Phases

  1. Foundation — content data model + translations.ts + LanguageService
  2. Shell — routing, LayoutComponent, HeaderComponent, FooterComponent
  3. Landing page — news items with HTML links
  4. Intro page — bio with floating image
  5. CV page — TimeRangePipe + section/subsection layout
  6. Works + Archive — NgbCarousel with captions
  7. Other page — static content with PDF/video links
  8. Assets — download images from lk2 repo to public/
  9. Accessibility pass — WCAG AA, focus management, ARIA
